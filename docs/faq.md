# FAQ (Frequently Asked Questions)

### Q: Why does the clock spin wildly in the Nether or The End?

Minecraft dimensions like the Nether and The End do not have a standard day-night cycle, meaning in-game time has no meaning there. The plugin detects this dimension state and sends a warning to the player.

### Q: How do I change the time format to 24-hour?

The plugin currently only supports the 12-hour format out of the box.

### Q: Does this support custom resource packs?

Yes. Since translations are handled server-side, players see the translated text based on their Minecraft client language preferences without needing custom packs.
