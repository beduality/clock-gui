# Permissions Reference

ClockTime uses a single permission node to control access to the clock feature.

## Reference Nodes

### `clock_time.use`

| Detail | Description |
|---|---|
| **Node** | `clock_time.use` |
| **Description** | Allows players to trigger the time display message by right-clicking a clock. |
| **Default Assignment** | `true` (Granted to all players) |
| **Type** | Boolean |
