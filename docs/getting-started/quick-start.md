# Quick Start

Get up and running in under a minute.

## 1. Install the plugin

Download the latest `.jar` from [GitHub Releases](https://github.com/beduality/clock-time/releases) and drop it into your server's `plugins/` folder.

## 2. Restart the server

```bash
# Or use /restart if your server supports it
```

## 3. Use it

1. Hold a **Clock** :material-clock: in your main hand.
2. **Right-click**.
3. The in-game time appears in chat, formatted in your client language.

!!! example "Output"

    <span style="color: #ffaa00;">Current Time:</span> <span style="color: #55ffff;">2:30 PM</span>

## What Happens in the Nether or End?

In dimensions where time doesn't exist, the clock displays a special message instead:

!!! quote ""

    <span style="color: #ff5555;">The clock spins wildly... Time has no meaning here.</span>
